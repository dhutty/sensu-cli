sensu-cli
===============================

version number: 0.1.0
author: Duncan Hutty

Overview
--------

A convenient CLI for some of the Sensu [REST API].

[REST API]: https://sensuapp.org/docs/0.28/api/

Installation / Usage
--------------------

To install use pip:

    $ pip install sensu-cli

Or clone the repo:

    $ git clone https://github.com/dhutty/sensu-cli.git
    $ python setup.py install
